# Persoal Profile Web

Personal Profile Web / Blog / Activity / Project / Resume / Contract.


## Setup Project

### 1. Use Node Version
* node version 20.11.1

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Project
```bash
npm run dev
```
